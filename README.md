# Candle Flicker

Programmable LED candle with realistic flame simulation.

## Features
- Realistic candle flicker algorithm
- Automatic dusk / dawn detection
- 3D printable flame diffuser
- Arduino / ATTiny compatible

## Hardware
- Arduino Uno (prototype)
- NeoPixel LED
- Photoresistor
- 3D printed flame diffuser
